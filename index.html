Well well well, Looks like it works. Now being served by browser-sync.
Remember here's how you launch Browser-sync from the terminal:
$ browser-sync start --server --d --f "**/*"
To install browser-sync from npm use:
$npm install -g browser-sync
(-g installs for all users)
You might also have to install gyp (some python thing) $npm install -g node-gyp
